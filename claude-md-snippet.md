## Accessibility

All generated UI must meet WCAG 2.2 AA. Rules and required verification: @accessibility.md
All other generated artifacts (PDF, docx, pptx, xlsx, email, markdown, images) follow @output-accessibility.md

No page is "done" until the five checks in that file pass. If a design request conflicts with those rules, flag the conflict and propose an accessible alternative before building.
