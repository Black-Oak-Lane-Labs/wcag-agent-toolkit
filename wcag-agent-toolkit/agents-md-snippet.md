## Accessibility

All generated UI must meet WCAG 2.2 AA. Read and follow ./accessibility.md in full, including the five verification steps at the end. All other generated artifacts (PDF, docx, pptx, xlsx, email, markdown, images) follow ./output-accessibility.md.

Failing any of those checks means the task is incomplete. If a design request conflicts with the rules in that file, flag the conflict and propose an accessible alternative before building.
